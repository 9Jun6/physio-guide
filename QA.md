## 미수정 이슈 (낮은 우선순위)

### ⚠️ 운동 목록/상세 페이지 — 네트워크 오류 시 무한 로딩

**파일**: `app/exercises/page.tsx`, `app/exercise/[id]/page.tsx`
**조건**: `/api/exercises` 호출 실패 시 `.catch()` 핸들러 없음 → `loading=true` 영구 유지
**영향**: 정상 네트워크 환경에서는 발생하지 않음. 로컬 개발/배포 환경 모두 안정적이면 무시 가능.

### ℹ️ 처방전 GET API — 비밀번호 쿼리 파라미터 노출

**파일**: `app/api/prescriptions/route.ts`
**내용**: 관리자 패널이 `?adminPassword=...` 형태로 GET 요청 → 서버 액세스 로그에 비밀번호 기록 가능
**영향**: 소규모 단일 사용자 앱이므로 즉각적인 위협은 낮음. 보안 강화 시 POST 또는 헤더 방식으로 교체 권장.
